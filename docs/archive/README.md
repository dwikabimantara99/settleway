# Documentation Archive

This folder preserves historical Settleway material that is no longer the active source of truth.

## Folders

- `acceptance/` - historical phase acceptance reports.
- `handoffs/` - previous onboarding, handoff, execution, and salvage documents.
- `phases/` - historical phase plans and scope gates.
- `prompts/` - original AI execution prompts.
- `testnet/` - historical Testnet runbooks, readiness notes, and smoke evidence.

Use these files for context and provenance only. Current implementation work must begin with `docs/active/`.
